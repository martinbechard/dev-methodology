Agent and Skill Definitions

This page owns generated agent and skill definitions, including responsibilities, assigned skills, output contracts, examples, model profiles, repository mutation policies, and agent-skill relationships. The interactive diagram and the cards deliberately present the same canonical relationships in two generated views; no hand-authored routing policy lives here. Role Agent Categories

The role model has four operating categories: agents used during normal development, agents that own wiki activities, agents that set up or update a downstream project, and agents that maintain this methodology bundle.

Amber marks a fixed role-skill edge, violet marks a conditional role-skill edge, and blue identifies a setup-time technology skill in the catalog. The specialization strategy owns when and how those skills are selected.
