# Dev Security Reviewer Suite Contract

Evaluate read-only attack-path analysis grounded in actual code and configuration. A finding must connect an untrusted input or actor through trust and authorization boundaries to protected data or side effects, cite exact evidence, and be prioritized by exploitability and impact. A clean result must identify reviewed boundaries and residual risk. Unavailable governing authority limits the conclusion rather than inviting assumptions. Judge trace quality, evidence, prioritization, uncertainty discipline, and actionable residual risk.
