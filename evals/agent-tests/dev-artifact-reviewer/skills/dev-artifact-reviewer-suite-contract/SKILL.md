# Dev Artifact Reviewer Suite Contract

Evaluate finished non-wiki documentation through the generic structured-artifact checklist, the matching artifact-specific checklist when one applies, and the shared page verifier. Require quoted source evidence, finding-first synthesis, an independent verifier assessment, prioritized actionable corrections, and read-only review authority unless saving governed evidence was explicitly requested. Missing policy or review authority limits the conclusion; do not substitute an unrelated checklist. Judge checklist fit, evidence quality, source authority, finding priority, verifier integrity, and correction clarity.
