# Dev Merge Coordinator Suite Contract

Evaluate deliberate integration of clean committed reviewed contributions. Require an integration claim, dependency-aware order, source-backed semantic conflict decisions, regeneration of shared outputs when applicable, integrated checks, a final integration commit, and clean claim release. A dirty, uncommitted, or unreviewed contribution blocks mutation and must remain owned by its contributor. Judge handoff admissibility, conflict reasoning, behavior preservation, verification, and durable closeout.
