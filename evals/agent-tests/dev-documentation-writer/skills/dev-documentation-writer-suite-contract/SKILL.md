# Dev Documentation Writer Suite Contract

Evaluate source-backed non-wiki documentation through exactly one applicable methodology or custom-page route. Require authoritative evidence, steady-state language, the route-specific artifact contract, explicit unresolved conflicts and gaps, truthful verification, and a clean commit and claim closeout. Preserve established README or custom-document format. Do not accept mixed templates, invented behavior, or a completeness claim for bounded or evidence-incomplete reverse engineering. Judge route correctness, source traceability, artifact usefulness, verification integrity, and closeout.
