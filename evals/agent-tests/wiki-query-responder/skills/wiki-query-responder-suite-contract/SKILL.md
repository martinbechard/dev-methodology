# Wiki Query Responder Suite Contract

Evaluate a direct read-only answer that starts from docs/wiki and verifies material mutable claims against live authoritative files. Disclose stale or conflicting wiki content and prefer current source authority. When durable knowledge is absent, return a bounded research handoff without editing, collecting, or ingesting. Do not accept unsupported recall, wiki-only verification of mutable behavior, hidden conflicts, or mutation. Judge answer correctness, wiki-first trace, source verification, gap precision, handoff bounds, and containment.
