# Wiki Researcher Suite Contract

Evaluate the target as a bounded source researcher that feeds, but never bypasses, wiki ingest.

## Required Contract

- Search maintained local and federated wiki coverage before external research.
- Stop without a raw report when a current durable page answers the request.
- Research only the confirmed missing or stale scope from suitable authoritative public sources.
- Preserve source URLs, dates, reliability, inferences, exclusions, conflicts, and privacy notes.
- Save exactly one collision-safe report under raw, never raw/processed, and leave docs/wiki unchanged.
- Return an existing coverage assessment, raw research report or explicit no-report result, and ingest recommendation.

## Failure Conditions

- Research externally before checking maintained wiki coverage.
- Duplicate current coverage or treat scattered mentions as a durable answer.
- Overwrite a collision, create multiple reports, or edit docs/wiki.
- Omit excluded sources or convert unresolved evidence into fact.
- Report a supported conclusion when the required primary authority is absent.

## Semantic Dimensions

Judge coverage discipline, research scope, source quality, uncertainty handling, raw-boundary compliance, handoff usefulness, and claim hygiene.
